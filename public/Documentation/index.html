Overview

Unify is an incredibly beautiful responsive Bootstrap Template for corporate and creative professionals. It works on all main web browsers, tablets and phones. This version is v1.0 and much more new features are coming soon in the next releases. Thank you for using Unify.

Main Structures top

Template files consist of top, header, slider, content part, and footer

Beginning of Page

Below code is used at the beginning of all HTML pages to detect Internet Explorer browser version and set a spesific class applied to Internet Explorer versions.

<!DOCTYPE html>
<!--[if IE 8]> <html lang="en" class="ie8"> <![endif]-->
<!--[if IE 9]> <html lang="en" class="ie9"> <![endif]-->
<!--[if !IE]><!--> <html lang="en"> <!--<![endif]-->

Header & Navigation top

Unify Template consist in two basic navigations (menu) and you can use any of them in your project. Here has shown how you can connect navigations.

Moreover, do not forget to connect top as shown below:

<!--=== Top ===-->
<div id="top">
    <div class="container">
	    ...
    </div><!--/container-->
</div><!--/top-->
                
<!--=== Header ===-->
<div id="header">
    <div class="container">
	    ...
	    ...
    </div><!--/container-->
</div><!--/header-->

If you want to use an alternative menu, you should change in head the style's name also:

<head>
    <!-- This is default menu style -->
    <link href="assets/css/menu/style-menu1.css" rel="stylesheet" />
    
    <!-- OR (This is an alternative menu style) -->
    <link href="assets/css/menu/style-menu.css" rel="stylesheet" />
</head>

The default menu style is assets/css/menu/style-menu1.css

Fonts top

@import url(http://fonts.googleapis.com/css?family=Open+Sans);
